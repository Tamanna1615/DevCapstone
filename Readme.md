## Description

This repository is created as part of the **DevOps Capstone Project** course assignment.  
It demonstrates a complete DevOps development lifecycle, including project planning, issue tracking, Continuous Integration and Continuous Delivery (CI/CD), code quality checks, and security scanning.

The project follows industry best practices for **Agile development**, **automated testing**, and **secure software delivery** using GitHub-native tools.

---

## Project Details

- **Repository Type:** Public GitHub repository  
- **Methodology:** Agile / Scrum  
- **CI/CD Platform:** GitHub Actions  
- **Project Management:** GitHub Issues & GitHub Projects (Kanban)
